Para decir el precio...

4,50€ -> _cuatro euros cincuenta_ = **vier euro funfzig** 

Cuando el precio empieza por 0...

0,90 = _noventa céntimos_ = **neunzig zent**

Para decir cuánto cuesta algo...
**Das macht** = (literalmente _eso hace_)

_Das macht 2,50 €_

---
Algunos ejemplos de cómo decir y preguntar el precio:

**Was kosten die Chips?**
**Was kostet die Lampe?**

Es decir, el verbo debe ir conjugado con el sujeto.

También puede preguntarse como se hace en castellano (cuánto cuesta)

**Wie viel kosten die Chips?**
